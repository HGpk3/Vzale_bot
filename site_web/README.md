# Site Web (Frontend)

## Planned Stack
- Next.js (App Router)
- TypeScript
- TanStack Query
- Zustand (UI state)
- Tailwind (or design-system tokens)

## Planned Route Map
- `/login`
- `/dashboard`
- `/tournaments`
- `/tournaments/[id]`
- `/my/team`
- `/my/achievements`
- `/my/stats`
- `/admin` (protected)
- `/admin/tournaments/[id]`
- `/public/tournaments/[id]`

## Integration
The frontend consumes `site_backend` API (`/v1/*`).
